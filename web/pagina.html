TECNOLOGÍA

Cómo funciona ZBLZ Engine

Usa LD_PRELOAD para inyectar una biblioteca que intercepta todas las llamadas de tiempo del sistema.

📡

Intercepta llamadas del sistema

clock_gettime, gettimeofday, nanosleep, usleep, sleep y más.

C → LD_PRELOAD → libspeedhack.so
⏱️

Modifica el tiempo percibido

El juego cree que el tiempo pasa más rápido o más lento, pero el sistema sigue normal.

Fórmula: modified_time = initial + (elapsed × speed)
🚀

Dos modos de uso

  • 1 Steam Launch Options (actual)
  • 2 Adjuntar a proceso en tiempo real (próximamente)
3 PASOS • 5 MINUTOS

Inicio Rápido

1

Construye la biblioteca speedhack

cd lib/
chmod +x build.sh
./build.sh install

Instala libspeedhack.so en ~/.local/lib/zblz/

2

Lanza la interfaz gráfica

cd scripts/zblz_engine
pip install -r requirements.txt
python main.py
3

Genera el comando para Steam

Ajusta el slider → “Generate Command” → pégalo en las propiedades del juego.

REQUISITOS

GCC + build-essential
sudo apt install build-essential
Opcional 32-bit
gcc-multilib

Ejemplo de comando generado

LD_PRELOAD="/home/usuario/.local/lib/zblz/libspeedhack.so"
SPEED=2.00 %command%
+ MangoHud
+ GameMode

Úsalo con cualquier juego de Steam

Incluyendo Proton y Wine
